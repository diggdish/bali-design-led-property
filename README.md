# Bali design-led property website

Static premium single-page website for a Bali property investment and retirement-lifestyle concept led by Andika Praba in collaboration with Dwik.

## Suggested image folder structure

```text
assets/
  portfolio/
    dezier-portfolio.pdf
    samara-bay.jpg
    samara-s7.jpg
    kiko-sejuk.jpg
    tba-villa.jpg
    wedding-chapel.jpg
    oblock-restaurant.jpg
```

Portfolio card content is grouped in `index.html` and marked with comments so final project names, permissions, or replacement images can be updated quickly.

## Run locally

Open `index.html` directly in a browser, or run a local server from this folder:

```bash
python -m http.server 5173
```

## Verify

```bash
npm test
```
